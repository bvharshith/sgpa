# VTU-SGPA-CGPA-CALCULATOR
An Interactive Website Made Using Technologies Such As Python, Node-Js, HTML, CSS, Javascript.
# System Requirements
* <b>Os :</b> Windows 10
* <b>RAM :</b> 4GB
* <b>Python version 3.0 or latest must be required</b>
